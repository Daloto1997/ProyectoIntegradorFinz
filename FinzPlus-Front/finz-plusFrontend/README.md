## 🛠️ Avance 2: Funcionalidad e Interactividad
- **Navegación:** Implementada con `react-router-dom`.
- **Protección de Rutas:** El Dashboard redirige al Inicio si no detecta sesión en `localStorage`.
- **Gestión de Estado:** Uso de `useState` para movimientos y `useEffect` para validación de sesión.
- **Componentes:** Los datos de gastos se envían a `ListaMovimientos` mediante **Props**.
